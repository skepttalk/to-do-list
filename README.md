# To-Do List Application

The To-Do List Application is a simple and interactive web application that helps users organize and manage daily tasks efficiently.

Users can add new tasks, mark tasks as completed, update existing tasks, and remove tasks when they are no longer needed. The application stores tasks locally in the browser, ensuring that tasks remain available even after refreshing the page.

This project demonstrates practical use of JavaScript for DOM manipulation, task management logic, and persistent storage using the browser’s Local Storage API.

## Project Overview

The goal of this project is to provide a lightweight task management system that allows users to track their daily activities.

The application dynamically updates the interface using JavaScript without requiring page reloads. Tasks are stored in Local Storage so that the user's data persists across browser sessions.

This project demonstrates fundamental frontend development concepts such as state management, dynamic DOM updates, and browser storage APIs.

## How the Application Works

1. The user enters a task in the input field.
2. JavaScript captures the input and saves the task to Local Storage.
3. The task is dynamically rendered on the page.
4. Users can mark tasks as completed or delete them.
5. All changes are automatically saved to Local Storage.
